Android finetune

1. What is does it do?
Top-notch professional fast tune up finetune android package. Tested with Nokia N900 (Maemo), Gionee Pioneer GN 100, Lenovo A536, Cubot Note S (Android 6).

1.2. Features
- faster system and data partition
- flexible (sysctl optimization)
- faster network
- can move dalvik and data folder (more internal space)

2. Requirements
- Android 4.4 - Android 6.0.1
- Busybox
- root

3. Installation
Download script and unpack it.

4. Tutorial
Open terminal and gain root. Navigate to script type sh tuneup.sh to start optimization. Use it at your own risk.

4.1 Dalvik-Cache and data optimization is disabled by default (symlink.sh and symlink2.sh).
